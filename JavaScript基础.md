# JavaScript

JS是一种脚本语言，用于页面交互，适配于所有现在的浏览器。



## 杂

- 注释

  - `//`  一行注释
  - `/*  */` 多行注释

  

## JS 八种数据类型

### undefined 未定义

### null 空值

### boolean  布尔值

### string  字符串

### symbol 符号

### BigInt  比2<sup>53</sup> -1 大的数值





### number  数值

1. 可用 `+`  `-` `*` `/` 计算

  ```js
  var sum = 10 + 10;
  var difference = 45 - 33;
  var product = 8 * 10;
  var quotient = 66 / 33;
  ```

> 支持小数及小数运算

2. 快捷

| 代码   | 意思         |      |
| ------ | ------------ | ---- |
| `i++;` | `i = i + 1;` |      |
| `i--;` | `i = i - 1;` |      |
|        |              |      |



### object

  

  ## 赋值

  `variable`

  命名可用数字，字母，`$`, `_` 命名，不可包含空格或用数字开头，命名区分大小写。建议的命名方式 `camelCase`，即第一个单词首字母小写，之后的单词首字母皆大写。

> 八种数据类型可能会被储存在变量里。

```js
var myVar;
myVar = 5;
```

**赋值初始值**

```js
var myVar = 0;
```
